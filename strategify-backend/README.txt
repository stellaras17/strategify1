Please assure you have python and pip installed on your local machine.

Download the ta-lib wheel according to your version of python from: 
https://www.lfd.uci.edu/~gohlke/pythonlibs/#ta-lib

Move the file to this directory, open a new terminal and run:

pip install <filename>

Then, install the rest required libraries by running:

pip install -r requirements.txt

After everything has installed, run on the terminal:

python main.py
